Make your HTML look like Markdown

The text you’re reading right now is not plain text, but simple HTML styled like Markdown text. Right-click and inspect this page to get a feel of how it works.

Getting started

Using ReMarkdown is as simple as downloading remarkdown­.css, and using a single class: 

<!doctype html>
<html lang="en">
<head>
  <title>Using ReMarkdown</title>
  <link rel="stylesheet" href="https://fvsch.github.io/remarkdown/dist/remarkdown.css">
</head>
<body class="remarkdown">
  <h1>Hello World</h1>
  <p>A paragraph.</p>
</body>
</html>

Modularity

ReMarkdown is broken down into a list of styles, and the default remarkdown.css uses a dozen of them by default.

You can choose additional variants by adding option names as classes along the remarkdown class. For example: 

<body class="remarkdown h1-underline pre-ticks">
  ...
</body>

If for some reason you don’t like the default options, you can use the ReMarkdown-zero builds which have no default styles, and list all the styles you want to use: 

<head>
  <link rel="stylesheet" href="https://fvsch.github.io/remarkdown/dist/remarkdown-zero.css">
</head>
<body class="remarkdown hn-reset hn-hash a-bracket ul-plus ol-alpha …">
  ...
</body>

For even greater control, you can build a custom ReMarkdown stylesheet using Sass.

Technicalities

  • Copyright: the MIT License applies (ReMarkdown 2.1 and older were licensed under the WTFPL).
  • Browser support: I’m targetting the most current browsers (mid-2014). Theoretically it should work on IE9+, but that’s a very much untested assertion.
  • This is no HTML-to-Markdown converter. Try Pandoc.
