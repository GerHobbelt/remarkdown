Make your HTML look like Markdown

The text you’re reading right now is not plain text, but simple HTML styled like Markdown text. Right-click and inspect this page to get a feel of how it works.

But, why?!?

Because it was fun to do.

Usage

Using ReMarkdown is as simple as downloading remarkdown­.css, and using a single attribute: 

<!doctype html>
<html lang="en">
<head>
  <title>Using ReMarkdown</title>
  <link rel="stylesheet" href="remarkdown.css">
</head>
<body data-remarkdown>
  <h1>Hello World</h1>
  <p>A paragraph.</p>
</body>
</html>

Modularity

ReMarkdown is broken down into a list of styles, and the default remarkdown.css uses a dozen of them by default.

You can choose additional variants by adding option names in the data-remarkdown attribute. For example: 

<body class="remarkdown h1-underline pre-ticks">
  ...
</body>

And there’s a bunch of alternative styles to pick from! Define your prefered styles using the data-rmd attribute, or when compiling from the Sass source (see the configuration guide for details).
